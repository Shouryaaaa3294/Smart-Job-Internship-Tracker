# Smart Job Tracker

A full-stack MERN application to track job and internship applications with Kanban workflow and analytics.

## Features

* Track job applications
* Kanban board workflow
* Authentication (JWT)
* Dashboard analytics

## Tech Stack

* React, Tailwind CSS
* Node.js, Express
* MongoDB

## Installation

```bash
git clone https://github.com/yourusername/job-tracker.git
cd job-tracker
npm install
```

## Run

```bash
npm run dev
```
